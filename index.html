Overview

Our goal is to use Google's Tensorflow framework to create a Deep Learning Convolutional Neural Network (CNN) model for license plate detection.

The app features a trained model that detects license plates and license plate numbers in images/videos. Using the license plate number, it then queries a database for information and displays it to the user.
